# 学習支援アプリ

中学生向けの問題演習 Web アプリです。問題を JSON ファイルで登録し、4択・記述形式で練習できます。苦手な問題ほど優先的に出題されるため、効率よく実力を伸ばせます。

**公開 URL:** https://ken-ji.github.io/study-app/

---

## 生徒向け：使い方

### 1. アプリを開く

ブラウザで https://ken-ji.github.io/study-app/ を開いてください。スマートフォンでも利用できます。

### 2. 演習を始める

**「5問モード」または「10問モード」** を選んでボタンをタップします。

> 問題データがない場合はボタンが押せません。先生に問題ファイルを受け取って読み込んでください（→「問題ファイルの読み込み」参照）。

### 3. 問題に答える

- **4択問題** — 選択肢が4つ表示されます。正しいと思う答えをタップしてください。
- **記述問題** — 入力欄に答えを入力して「回答する」を押してください。Enterキーでも回答できます。

回答すると正誤とともに**解説**が表示されます。「次の問題へ」で進んでください。

### 4. 結果を確認する

全問回答後に**結果画面**が表示されます。

- 正解数・正答率が確認できます
- 間違えた問題には「あなたの回答」「正答」「解説」が一覧で表示されます

「トップに戻る」でホームに戻り、もう一度演習を始められます。

### 5. 苦手を確認する

ナビゲーションバー（画面上部）の **「苦手分析」** を開くと、演習の積み重ねに応じて以下が確認できます。

| 表示内容 | 説明 |
|---------|------|
| 単元別正答率 | 単元ごとの出題数・正解数・正答率 |
| 苦手問題一覧 | 正答率が低い問題を順番に表示 |

> 正答率が低い問題ほど次の演習で出題されやすくなっています。

## 先生・保護者向け：セットアップと管理

### 問題ファイルの準備

問題は **JSON ファイル** で作成します。テキストエディタ（メモ帳など）で作成し、`.json` という拡張子で保存してください。

#### ファイル形式（例）

```json
[
  {
    "id": "MATH101",
    "unit": "正の数・負の数",
    "type": "choice",
    "question": "(-3) + (-5) を計算せよ",
    "answer": "-8",
    "explanation": "負の数どうしの加法は、絶対値を足して負の符号をつける。3 + 5 = 8 なので答えは -8"
  },
  {
    "id": "MATH201",
    "unit": "一次方程式",
    "type": "text",
    "question": "2x + 4 = 10 を解け",
    "answer": "3",
    "explanation": "2x = 10 - 4 = 6、よって x = 3"
  }
]
```

#### 各フィールドの説明

| フィールド | 必須 | 説明 |
|-----------|------|------|
| `id` | ✅ | 問題の一意な識別子。重複するとインポート時にスキップされます |
| `unit` | ✅ | 単元名（例：`"一次方程式"`）。苦手分析の分類に使われます |
| `type` | ✅ | `"choice"`（4択）または `"text"`（記述）|
| `question` | ✅ | 問題文 |
| `answer` | ✅ | 正答。記述問題では全角・半角・前後スペースを無視して採点されます |
| `explanation` | ✅ | 解説文。回答後に表示されます |

> **4択問題（`type: "choice"`）の選択肢について**  
> 選択肢は自動生成されます（正答 + 誤答3つ）。数値の答えには近い値が誤答として生成されます。手動で選択肢を指定することはできません。

#### ファイルの構造は2通り使えます

```json
// パターン1：配列をそのまま書く（上記の例）
[{ ... }, { ... }]

// パターン2：questions キーに入れる
{ "questions": [{ ... }, { ... }] }
```

---

### 問題ファイルの読み込み

1. アプリのトップ画面を開く
2. **「JSONを読み込む」** ボタンをクリック
3. 作成した JSON ファイルを選択する
4. 「○問追加、○問スキップ（重複）」というメッセージが出れば成功です

> 同じ `id` の問題は2回目以降のインポートでスキップされます。問題を更新したい場合は先に削除してから再インポートしてください。

---

### 問題の管理・削除

ナビゲーションバーの **「問題管理」** から操作します。

1. **単元で絞り込む** — ドロップダウンで単元を選ぶと、その単元の問題だけ表示されます
2. **問題を選択する** — 各行のチェックボックスをクリック（左端のチェックボックスで全選択）
3. **削除する** — 「削除」ボタンをクリックし、確認ダイアログで「削除する」を選択

> 削除は元に戻せません。誤って削除した場合は JSON ファイルを再インポートしてください。

---

### データについて

- 問題データ・学習履歴はすべてブラウザの **localStorage** に保存されます
- サーバーへの送信や外部への共有は行いません
- **ブラウザのデータを消去すると、問題データも履歴もすべて削除されます**
- 複数の端末でデータを共有することはできません

## よくある質問

**Q. 問題が表示されない / ボタンが押せない**  
問題データがインポートされていません。「JSONを読み込む」から問題ファイルを読み込んでください。

**Q. 記述問題で正しい答えを入力したのに不正解になる**  
答えに全角・半角の違いがある場合も自動的に対応しています。問題の `answer` フィールドの値と大きく異なる場合は、問題ファイルの `answer` を確認してください。

**Q. スマートフォンで使えますか？**  
Chrome（Android）・Safari（iOS）で動作します。ホーム画面に追加すると、アプリのように使えます。

**Q. 問題を追加したら以前の学習履歴はどうなりますか？**  
履歴はそのまま残ります。新しく追加した問題には履歴がないため、最初は出題確率が高くなります。
